OCR PDF text recognition

OCR PDF is for scanned documents and image-only pages that need text recognition or searchable PDF workflows.

Scanned PDFs

A scanned PDF is usually a picture of text, so normal PDF to Word conversion cannot make the text editable without OCR.

Production requirement

A production OCR worker such as Tesseract, Google Vision, Azure OCR or AWS Textract is required before this tool can process scanned documents.

How to use this page

  1. Upload a scanned PDF.
  2. Run OCR when the OCR worker is configured.
  3. Use the recognized text for search or editing workflows.
