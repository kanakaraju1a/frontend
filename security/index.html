FileFlow security

Security matters because file conversion tools can process sensitive documents and images.

Upload limits

Production hosting should enforce file size limits, MIME validation, HTTPS and rate limits.

Temporary files

Backend conversion jobs should use temporary folders and delete files after each request.

How to use this page

  1. Use HTTPS pages.
  2. Avoid unsupported files.
  3. Delete downloaded outputs when no longer needed.
