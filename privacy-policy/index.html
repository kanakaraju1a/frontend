FileFlow privacy policy

FileFlow is designed to minimize file exposure and explain how browser and backend conversion workflows handle files.

Browser tools

Many image and PDF utility tools run locally in the browser.

Backend conversion

Word and advanced PDF conversion use the configured backend service and should remove temporary files after each request.

How to use this page

  2. Review whether the tool uses browser or backend processing.
  3. Contact support for privacy questions.
